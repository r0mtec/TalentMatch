# TalentMatch Frontend

Frontend-приложение на Vite + React + JavaScript. Backend пока не подключается: все запросы, кандидаты и справочник технологий работают через mock-состояние в `src/services/mockApi.js`.

## Запуск

```bash
npm install
npm run dev
```

Приложение откроется на адресе, который выведет Vite. Экран входа доступен по `/login`, рабочие экраны открываются через React Router.
